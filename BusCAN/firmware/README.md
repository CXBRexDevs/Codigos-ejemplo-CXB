# Hardware
Para el uso de este protocolo es necesaria la implementación de dos transceptores CAN "TJA1050" conectando uno de ellos a los "CAN_TX" y "CAN_RX" que corresponden a los pines 58 ->"C2TX" y 59->"C2RX".
